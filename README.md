## M3u8Downloader

> A m3u8 streaming spider and downloader. 一个m3u8流媒体的爬虫和下载器。

### 特性

* 爬取站点m3u8视频流地址
* 提取并合成视频流为mp4

### 使用

1. 安装python3（python2未测试）

2. 安装相应python依赖包

```
pip install -r requirements.txt
```
2. 命令行方式下载

```
python cmd.py http:xxx.com
```

3. gui方式
```
python gui.py
```
